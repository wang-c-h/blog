# 并查集  
在计算机科学中，并查集（英文：Disjoint-set data structure，直译为不交集数据结构）是一种数据结构，用于处理一些不交集（Disjoint sets，一系列没有重复元素的集合）的合并及查询问题。 并查集支持如下操作： 查询：查询某个元素属于哪个集合，通常是返回集合内的一个“代表元素”。  
## 合并（union）  
把两个不相交的集合合并为一个集合。
## 查询（Find）  
查询两个元素是否在同一个集合中。
